# Простое веб-приложение для управления БД пользователей с авторизацией, списком, созданием и редактированием записей


# Стек технологий:

● TypeScript 

● Webpack + Npm

● React 16

● Ant design 5 и styled-components

● React router 6

● TanStack Query 4 + axios

● dayjs

● Архитектура FSD (Feature-Sliced Design)
